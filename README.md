# github-notes
## Git
* Is your version control/sofware.
## Github
* Is your web service, where you do `Git` things on a server or website.
## Repo 
* Is a repository, basically like a project folder where you store your files dedicated to that project
## Commit
* Your commit saves your changes you've done to a file, and in that commit, theres a hash, where it shows your version histories and the changes you've committed.
